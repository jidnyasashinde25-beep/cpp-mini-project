# Hashing with File Handling in C++

## 📌 Description
This project demonstrates file handling using C++.  
It writes user input to a file, applies a simple hash function, and reads the data back.

## ⚙️ Features
- Write data to file
- Read data from file
- Simple hashing function

## 💻 Language
C++

## ▶️ Output
Enter name: jidnyasa
Data written to file  
Reading from file:  
jidnyasa -> 1
